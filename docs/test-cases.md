| case_id | 用户操作 | 输入内容 | 预期结果 | 是否应成功 | 说明 |
|---|---|---|---|---|---|
| N-01 | 新增一条完整知识 | 标题、正文、分类、标签、来源、链接均合法 | 条目保存成功，首页和列表页可见 | 是 | 正常新增主流程 |
| N-02 | 新增知识并加入收藏 | 标题、正文合法，勾选收藏 | 保存成功，详情页显示已收藏 | 是 | 验证收藏初始状态 |
| N-03 | 在首页搜索已有关键词 | 关键词命中标题 | 返回匹配条目 | 是 | 正常搜索流程 |
| N-04 | 在详情页切换归档 | 已存在条目 | 状态更新成功，UI 显示已归档 | 是 | 正常状态变更 |
| N-05 | 打开今日回顾页面 | 已存在带 `reviewAt` 的条目 | 页面显示今日待回顾卡片 | 是 | 正常回顾功能 |
| B-01 | 标题刚好 1 个字符 | 标题为单字，正文正常 | 如果未设最小标题长度则可保存 | 是 | 标题边界 |
| B-02 | 正文刚好达到最小长度 | 正文 10 个字符左右 | 允许保存 | 是 | 正文下限边界 |
| B-03 | 标签数量为 0 | 标题正文正常，无标签 | 允许保存，但提示建议补充标签 | 是 | 标签可选策略 |
| B-04 | 分类为空 | 标题正文正常，不选分类 | 允许保存，但提示建议补充分类 | 是 | 分类可选策略 |
| B-05 | 搜索不存在的长关键词 | 输入随机长词 | 页面显示空状态，不报错 | 是 | 无结果边界 |
| E-01 | 新增知识时标题为空 | 标题为空，正文正常 | 拦截提交，提示填写标题 | 否 | 必填项异常 |
| E-02 | 新增知识时正文过短 | 标题正常，正文少于最小长度 | 拦截提交，提示正文过短 | 否 | 内容质量异常 |
| E-03 | 来源链接格式错误 | `abc` 或 `www.test` | 拦截提交，提示链接格式不合法 | 否 | URL 校验异常 |
| E-04 | AI 摘要生成失败 | 模拟接口超时或返回非法 JSON | 页面提示 AI 提炼失败，原始内容不丢失 | 否 | 辅助能力异常 |
| E-05 | 删除知识后列表未更新 | 删除后不主动刷新状态 | 应触发列表刷新；若失败需给出提示 | 否 | 列表同步异常 |
| E-06 | 收藏状态切换失败 | 模拟存储失败 | 状态回滚，提示更新失败 | 否 | 收藏异常 |
| E-07 | 归档状态切换失败 | 模拟存储失败 | 状态回滚，提示更新失败 | 否 | 归档异常 |
| E-08 | 详情页访问不存在的 id | 手动输入错误路由参数 | 展示“知识条目不存在”页面 | 否 | 路由异常 |
| E-09 | 本地存储数据损坏 | `knowledge_items` 不是合法 JSON | 页面降级为空列表，不崩溃 | 否 | 数据异常 |
| E-10 | 回顾页没有任何候选条目 | 没有 `reviewAt` 且数据很少 | 各分区显示空状态文案 | 是 | 空数据回顾场景 |
